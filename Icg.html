Image Captioning with NSFW filter

My current thesis project based on Deep Learning.
Technologies used: Python, Keras backend with TensorFlow, Google Colab

Looking at a picture of a cat and recognizing it as one is an effortless task for humans. For a computer, the image is just a representation of numbers. To make sense of an image as a high-level object and describing it in natural language is a daunting task for a computer. This project aimed at creating an NSFW filter that separated unsafe images and generated captions for the safe images. The project was able to achieve this by using a pretrained convolutional neural network for the task of NSFW image classification. For image captioning, the project used a CNN+RNN merged model that generated captions in English. The project was able to achieve an accuracy of 89.96% in classifying images and produced meaningful captions for the safe images.

What did the project acheive?

  • Used multiple new technologies like TensorFlow, Keras and Google Colab
  • Learned multiple preprocessing techniques such as Image Normalization and Augmentation
  • Generated an Image captioning model based on the paper 'Show and Tell'(Vinyals et al,2015)
  • Learned how to use pretrained models and perform transfer learning
  • Created an Image classifying CNN using pretrained VGG16 base layers which could classify safe and NSFW images with 90% accuracy
  • Combined the two models to create a model that generates captions for safe images
