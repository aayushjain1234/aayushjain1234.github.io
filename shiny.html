Uber Data Analysis in R

Analyze Uber data from New York to understand patterns and trends of Uber pickups

Data storytelling is an important component of Machine learning which help an organisation to take informative decisions based on evidence With the help of visualizations a company can benefit by understanding complex information in graphical representation and take responsible decisions. In this project we use R to perform data analysis on Uber data an plot multiple graphs to demonstrate various information such as trips per day, trips in most frequent locations, plotting heatmaps, etc. The project makes use of multiple packages like dplyr, ggplot, tidyr to help make this process easier and visually appealing.

What did the project acheive?

  • Made use of multiple data exploratory packages in R like dplyr, tidyr etc
  • Illustrated multiple insights using bar graphs and heatmaps using ggplot2
  • Gained insights from plots created in R
