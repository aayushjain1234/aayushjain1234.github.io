Python Visualization
Dimensionality Reduction Algorithms (PCA+t-SNE)

An attempt to find insights and clusters in 2 multi-dimensional datasets containing information regarding bank clients and forest patches of Canada

The Project presented a Data Visualization problem based on clustering and classification techniques and the aim was to derive valuable insights. Two datasets were used namely patches and clients, of which Patches had 7 variables and 15120 records while clients had 9 variables and 43193 records. Firstly, the project applied dimensionality reduction technique to visualize high dimensional data. As the dataset had more than 2 dimensions dimensional reduction technique called as Principal Component Analysis (PCA) was used. PCA is used to find low dimensional representation of high dimension data so that it contains as much variation as possible. But for the second dataset as the PCA first component is less than 70%, it didnâ€™t contain much variation. Also the dataset was non-linear, Thus another algorithm called t-SNE was used. T-SNE (T-Distributed Stochastic Neighbor Embedding) was used to project non-linearly data into lower dimensions in a way that distance between points (neighbors) in lower dimensions reflected the distances between points in higher dimension.

What did the project acheive?

  • Applied PCA algorithm to reduce dimentionality of dataset and visualised insights
  • Applied t-SNE algorithm on non linear dataset and visualised insights
