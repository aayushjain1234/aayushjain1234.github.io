Weather Analysis Using Python

EDA, Data Preparation,Comparing and Modelling a Regression and Classification Model

The objective of this project was to select an appropriate dataset that could be used to create machine learning models that perform classification and regression in Python using the Scikit learn library. Scikit learn is a very popular python library which consists of many algorithms and methods that make machine learning programming easier. It has multiple supervised and unsupervised algorithms along with functions for other machine learning operations like splitting training and test data, calculation of performance metrics, etc. In this project, supervised algorithms for regression and classification was implemented on the data selected to demonstrate how efficient and easy Scikit learn can be.

What did the project acheive?

  • Worked with real world weather data from data.gov
  • Performed inital EDA using Python
  • Preprocessed the data and cleaned the data
  • Compared multiple machine learning models for classification like logistic Regression, Decision Tree Classifier, K nearest neighbour and Support vector classifier
  • Compared multiple machine learning models for regression like Linear Regression, Decision Tree Regressor, LASSO and Support vector machine
  • Developed a classification model with 71.2% accuracy and a regression model with 82.2% accuracy
